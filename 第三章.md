### 3.1.1 函数是第一类对象

> JavaScript 中函数拥有对象的所有能力
>
> - 通过字面量创建
> - 赋值给变量，数组项或其他对象的属性
> - 作为函数的参数来传递
> - 作为函数的返回值
> - 具有动态创建和分配的属性

> 函数也是对象，唯一的特殊之处在于它是可调用的（invokable），即函数会被调用以便执行某项动作

---

### 3.1.2 回调函数

> 即将函数作为另一个函数的参数，随后通过参数来调用该函数

```javascript
const values = [0, 5, 2, 4, 3, 8, 1]
values.sort(function (a, b) {
  return a - b
})
```
