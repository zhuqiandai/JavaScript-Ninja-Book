## 优质的测试

1. 可重复性

   可重复性需要确保测试不依赖于外部因素，如网络或 CPU 负载。

2. 简易性

   我们的测试应该只关注测试这一件事

3. 独立性

   应该单独执行测试。应该避免测试之间的依赖
